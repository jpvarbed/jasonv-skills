# adversarial-review

Run a fast two-persona or deep four-persona adversarial council across
live-verified agent seats configured on this device. The ignored
`council-config.json` records non-secret auth/setup, model, profile bindings,
and separate smoke/review availability. `fast` uses two low-effort
calls; `deep` uses four medium-effort calls. A built-in registry owns all
read-only invocations and structured checks. Gemini and unplanned runtime
substitution are prohibited.

Part of [jasonv-skills](../../AGENTS.md) — skills for AI coding agents. Install with the repo-root `install.sh`; when a task matches, read this `SKILL.md` and follow it.
