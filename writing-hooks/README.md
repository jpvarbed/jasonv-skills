# writing-hooks

Personal compilation of hook and social-post writing gotchas (X/LinkedIn) in Jason''s voice. Trigger on "write me a hook", "clean up this tweet", "draft a thread", "does this sound like me", or any time a tweet/thread/hook is being written or edited. Use alongside avoid-ai-writing and the motion hook skills (hook-writing, hook-tactics, hook-voice-patterns) — this catches the tells those miss and enforces a plain, finding-first voice. NOT for long-form articles (use edit-article / writing-shape) or de-AI-ing general prose (avoid-ai-writing alone covers that).

A skill for AI coding agents (Claude Code, Cursor, etc.). Drop it in your agent's skills directory — see the repo root `install.sh`.
