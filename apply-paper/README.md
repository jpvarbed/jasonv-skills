# apply-paper

Turn a research paper's finding into a concrete change to our skills/agents — distill the claim, map it to where it applies, design the change, and prove it helps. Use when the user says "apply this paper", "what would <paper> change for us", "turn this finding into a skill change", or when triaging a paper in docs/papers.md from "interesting" to "shipped". NOT for summarizing a paper (just read it) or general literature review. Pairs with linting-and-scoring (score the result), adversarial-review (red-team the change plan), and goal-spec (if the change is big enough to dispatch).

A skill for AI coding agents (Claude Code, Cursor, etc.). Drop it in your agent's skills directory — see the repo root `install.sh`.
